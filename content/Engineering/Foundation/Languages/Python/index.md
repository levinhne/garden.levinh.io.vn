# Python Programming Language

Tài liệu và kiến thức về Python - ngôn ngữ đa năng cho backend, scripting và data science.

## 📚 Chủ đề chính

### Core Concepts
- [[Python-Functions|🔧 Functions]]: Định nghĩa, type hints, decorators
- **Data Structures**: Lists, dicts, sets, tuples
- **OOP**: Classes, inheritance, polymorphism
- **Modules & Packages**: Import system, virtual environments
- **Error Handling**: Try-except, custom exceptions

### Advanced Topics
- **Asyncio**: Async/await, event loops
- **Decorators & Context Managers**: Advanced patterns
- **Generators & Iterators**: Memory-efficient iteration
- **Type System**: mypy, Protocol, TypeVar
- **Metaprogramming**: Metaclasses, descriptors

### Backend Development
- **FastAPI**: Modern async web framework
- **Django**: Full-stack web framework
- **Flask**: Lightweight web framework
- **SQLAlchemy**: ORM patterns
- **Pydantic**: Data validation

### Testing
- **pytest**: Testing framework
- **unittest**: Standard library testing
- **Mocking**: unittest.mock, pytest-mock
- **Coverage**: Test coverage tools

## 🎯 Tại sao chọn Python?

### Ưu điểm
- 📖 **Readability**: Syntax rõ ràng, dễ học
- 🔧 **Versatile**: Web, scripting, data science, ML
- 📦 **Rich Ecosystem**: PyPI với 400k+ packages
- 🚀 **Fast Development**: Rapid prototyping
- 👥 **Community**: Large, active community

### Nhược điểm
- ⚡ **Performance**: Chậm hơn compiled languages
- 🧵 **GIL**: Global Interpreter Lock hạn chế concurrency
- 📱 **Mobile**: Không phổ biến cho mobile apps
- 💾 **Memory**: Memory usage cao hơn

| Feature | Python | Go |
|---------|--------|-----|
| **Syntax** | Simple, readable | Simple, verbose |

| **Error Handling** | Exceptions | Error values |
| **Use Case** | Scripting, web, data | Backend, systems |
