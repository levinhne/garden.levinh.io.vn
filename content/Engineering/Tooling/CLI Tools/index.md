# CLI Tools

## 🐚 Shell Enhancements
- **zsh + Oh My Zsh**: Shell customization
- **Starship**: Cross-shell prompt
- **fzf**: Fuzzy finder
- **ripgrep (rg)**: Fast text search
- **bat**: Better cat with syntax highlighting
- **exa/eza**: Modern ls replacement

## 🔍 Code Search & Navigation
- **fd**: Fast file finder
- **ag (The Silver Searcher)**: Code search
- **ctags/universal-ctags**: Code navigation

## 📝 Text Processing
- **jq**: JSON processor
- **yq**: YAML processor
- **sed, awk**: Text manipulation

## 🗄️ Database Tools
- **pgcli**: PostgreSQL CLI with autocomplete
- **mycli**: MySQL CLI with autocomplete
- **redis-cli**: Redis command-line interface

## 🌐 HTTP & API Tools
- **curl**: HTTP client
- **httpie**: User-friendly HTTP client
- **grpcurl**: gRPC command-line client

## 🐳 Container Tools
- **docker**: Container platform
- **docker-compose**: Multi-container orchestration
- **kubectl**: Kubernetes CLI
- **k9s**: Kubernetes TUI

## 📊 Monitoring & Debugging
- **htop**: Process viewer
- **ncdu**: Disk usage analyzer
- **tcpdump**: Network packet analyzer
- **strace**: System call tracer
