# Technical Vocabulary

## 🏗️ Software Architecture

### System Design
- **Scalability** /ˌskeɪləˈbɪləti/: Khả năng mở rộng
- **Throughput** /ˈθruːpʊt/: Thông lượng
- **Latency** /ˈleɪtənsi/: Độ trễ
- **Bottleneck** /ˈbɑːtlnek/: Điểm nghẽn
- **Load balancing**: Cân bằng tải
- **Horizontal scaling**: Mở rộng theo chiều ngang
- **Vertical scaling**: Mở rộng theo chiều dọc

### Design Patterns
- **Abstraction** /æbˈstrækʃn/: Trừu tượng hóa
- **Encapsulation** /ɪnˌkæpsjuˈleɪʃn/: Đóng gói
- **Inheritance** /ɪnˈherɪtəns/: Kế thừa
- **Polymorphism** /ˌpɑːliˈmɔːrfɪzəm/: Đa hình
- **Dependency injection**: Tiêm phụ thuộc
- **Inversion of control**: Đảo ngược điều khiển

## 🗄️ Database & Data

### Database Operations
- **Query optimization**: Tối ưu truy vấn
- **Index** /ˈɪndeks/: Chỉ mục
- **Transaction** /trænˈzækʃn/: Giao dịch
- **Deadlock** /ˈdedlɑːk/: Bế tắc
- **Replication** /ˌreplɪˈkeɪʃn/: Sao chép
- **Sharding** /ˈʃɑːrdɪŋ/: Phân mảnh dữ liệu

### Data Structures
- **Array** /əˈreɪ/: Mảng
- **Linked list**: Danh sách liên kết
- **Hash table**: Bảng băm
- **Binary tree**: Cây nhị phân
- **Graph** /ɡræf/: Đồ thị
- **Queue** /kjuː/: Hàng đợi
- **Stack** /stæk/: Ngăn xếp

## 🌐 API & Networking

### API Terms
- **Endpoint** /ˈendpɔɪnt/: Điểm cuối
- **Payload** /ˈpeɪloʊd/: Dữ liệu truyền tải
- **Authentication** /ɔːˌθentɪˈkeɪʃn/: Xác thực
- **Authorization** /ˌɔːθərəˈzeɪʃn/: Phân quyền
- **Rate limiting**: Giới hạn tốc độ
- **Idempotent** /aɪˈdempətənt/: Tính bất biến

### Protocol Terms
- **HTTP status codes**: 
  - 2xx: Success
  - 4xx: Client errors
  - 5xx: Server errors
- **gRPC** /ˈdʒiː ɑːr piː siː/: Google RPC
- **REST** /rest/: Representational State Transfer
- **WebSocket**: Giao thức song công

## 🐳 DevOps & Cloud

### Container & Orchestration
- **Containerization** /kənˌteɪnəraɪˈzeɪʃn/: Đóng gói container
- **Orchestration** /ˌɔːrkɪˈstreɪʃn/: Điều phối
- **Pod** /pɑːd/: Đơn vị nhỏ nhất trong Kubernetes
- **Namespace** /ˈneɪmspeɪs/: Không gian tên
- **Deployment**: Triển khai
- **Rolling update**: Cập nhật dần dần

### CI/CD Terms
- **Pipeline** /ˈpaɪplaɪn/: Đường ống CI/CD
- **Artifact** /ˈɑːrtɪfækt/: Sản phẩm build
- **Rollback** /ˈroʊlbæk/: Hoàn tác
- **Blue-green deployment**: Triển khai xanh-lam
- **Canary release**: Phát hành canary

## 💬 Common Phrases

### Code Review
- "LGTM" (Looks Good To Me): Trông ổn
- "Could you elaborate on...?": Bạn có thể giải thích thêm về...?
- "This approach seems reasonable": Cách tiếp cận này có vẻ hợp lý
- "Consider refactoring this part": Cân nhắc tái cấu trúc phần này
- "This might cause performance issues": Điều này có thể gây vấn đề về hiệu năng

### Meetings
- "Let's circle back to this": Hãy quay lại vấn đề này
- "Can you walk me through...?": Bạn có thể giải thích chi tiết...?
- "That makes sense": Điều đó có lý
- "I'll take an action item on this": Tôi sẽ xử lý việc này
